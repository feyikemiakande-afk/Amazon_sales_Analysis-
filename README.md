# Amazon Sales Performance Analysis (Power BI)

## Project Overview
This project analyzes Amazon sales data to monitor performance and identify trends. An interactive Power BI dashboard was built to track key metrics and support insight-driven analysis.

## Dataset
The dataset contains Amazon sales information, including order data, revenue, product categories, and time-based sales records.

## Tools Used
- Power BI
- DAX (basic measures)
- Data Modeling
- Data Visualization

## Key Metrics
- Total sales
- Total orders
- Average order value
- Sales trends over time

## Key Insights
- Sales performance varies across product categories
- Clear trends can be observed over time
- A small number of products contribute a large share of total revenue

## Conclusion
This project demonstrates Power BI skills, including data modeling, KPI development, and interactive dashboard design.
